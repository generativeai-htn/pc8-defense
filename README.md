# PC-8 Guardian Academy

สื่อการเรียนการสอนแบบเกม CAI สำหรับหน่วยที่ 8 “คลินิกสุขภาพคอมพิวเตอร์” วิชาระบบปฏิบัติการคอมพิวเตอร์ ครอบคลุม Check Disk, Optimize & Defragment, Disk Cleanup, Antivirus และ OneDrive ตามสไลด์ `Computer_Health_Clinic.pdf`

## รูปแบบการเรียนรู้

- ฝึกปฏิบัติผ่านระบบจำลอง ไม่ใช่แบบทดสอบเลือกตอบ
- Check Disk: เรียง Workflow และสแกนซ่อมเซกเตอร์เสีย
- Defragment: สลับบล็อกข้อมูลจนเหลือ 0% fragmented
- Disk Cleanup: คัดไฟล์ขยะพร้อมปกป้องเอกสารสำคัญ
- Antivirus: อัปเดตฐานข้อมูล เปิด Real-time protection และตั้งการสแกน
- OneDrive: ลงทะเบียน เชื่อม Sync, Upload, Share และ Free up space
- ทุกบทเชื่อมกับเกมป้องกันฐาน และจบด้วย Boss Rush ที่ต้องจับคู่อาการกับยูทิลิตี้ให้ถูกต้อง

ตัวเกมใช้ทีมแมว 15 ตัว ศัตรูปกติ 8 แบบ บอส 7 แบบ แผนที่ 5 ฉาก รวมถึง Cat Guardian และ Cat Boxing จาก Game Pack พร้อมเสียงดนตรีและเอฟเฟกต์

เว็บเป็น Static Site เปิดผ่าน GitHub Pages ได้โดยไม่ต้องติดตั้งแพ็กเกจ และบันทึกความก้าวหน้าไว้ในเบราว์เซอร์ด้วย `localStorage`
