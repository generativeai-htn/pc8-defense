# PC-8 Defense

เกมสื่อการเรียนรู้หน่วยที่ 8 “คลินิกสุขภาพคอมพิวเตอร์” สำหรับวิชาระบบปฏิบัติการคอมพิวเตอร์ ครอบคลุมเนื้อหา Check Disk, Optimize & Defragment, Disk Cleanup, Antivirus และ OneDrive

## เส้นทางการเรียนรู้

1. วิเคราะห์อาการและเลือกโปรแกรมยูทิลิตี้ให้ตรงปัญหา
2. เรียนรู้ขั้นตอนการใช้งานก่อนเข้าสนามต่อสู้แต่ละห้อง
3. วางหน่วยยูทิลิตี้เพื่อป้องกันเครื่อง PC-8 จากไฟล์ขยะ ไฟล์เสียหาย และไวรัส
4. ตอบคำถามเพื่อปลดล็อกห้องถัดไป
5. ต่อสู้กับบอส ทำภารกิจใบงานปฏิบัติ และจัดตารางดูแลสุขภาพเครื่อง

เกมทำงานแบบเว็บไซต์สแตติก เปิด `index.html` ผ่านเว็บเซิร์ฟเวอร์หรือ GitHub Pages ได้โดยไม่ต้องติดตั้งแพ็กเกจเพิ่มเติม ความคืบหน้าจะถูกบันทึกไว้ในเบราว์เซอร์ของผู้เล่น
